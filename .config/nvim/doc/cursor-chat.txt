*cursor-chat.txt*	Cursor agent chat (configs.cursor)

==============================================================================
*cursor-chat*

Lua:	lua/configs/cursor.lua
README:	lua/configs/README-cursor.md

Runs cursor-agent --print with stream-json into buffer cursor://chat.
Lazy require; no setup().

	:helptags ~/.config/nvim/doc
	:help cursor-chat

==============================================================================
COMMANDS						*cursor-chat-commands*

:Cursor {text...}	Send prompt.
:CursorStop		Stop job; add "> " (no stop_spinner).
:CursorToggle		Close or open split + prompt.
:CursorClear		Clear buffer; keep UUID.
:CursorNew		New UUID; clear buffer.
:CursorPick		Telescope resume.
:CursorMode		agent / ask / plan.

==============================================================================
LOADING						*cursor-chat-loading*

|BufEnter| on the chat buffer reapplies buffer-local keymaps.

==============================================================================
BUFFER						*cursor-chat-buffer*

Last line "> " — |hl-Question| on the prefix (prompt_ns, highlight).
Insert <CR> sends trimmed text after the prefix.

==============================================================================
STATUS-LINES					*cursor-chat-status-lines*

Tool and spinner rows use nvim_buf_set_extmark with line_hl_group
(Comment, or DiagnosticError for failed shell), priority 200, on
status_ns.  Whole line is highlighted.  Spinner row clears status_ns
before redraw.  Consecutive tool lines: trailing blank replaced when
the line above starts with "[".

Assistant append_text adds no status_ns marks.

==============================================================================
ASSISTANT					*cursor-chat-assistant*

Substring/replay skips; spinner line may be deleted; 300 ms idle may
restart "working" spinner when job_id is set.

==============================================================================
API						*cursor-chat-api*

require('configs.cursor'):
  send  stop  toggle  open_vertical  open_horizontal
  clear  new_chat  pick_chat  cycle_mode

==============================================================================
